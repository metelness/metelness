Project Overview

Built a system to detect if an individual was present during interactions, combining Python ML for prediction, Teradata SQL for data storage, and SAS for validation.

Tools Used

  • Python ML: Built a logistic regression model to predict presence
  • Teradata SQL: Stored and queried interaction data
  • SAS: Validated model outputs

Sample Python Code

import pandas as pd
from sklearn.linear_model import LogisticRegression

# Load data (simulated)
data = pd.read_csv("interaction_data.csv")
X = data[["interaction_duration", "signal_strength"]]
y = data["is_present"]

# Train model
model = LogisticRegression()
model.fit(X, y)
